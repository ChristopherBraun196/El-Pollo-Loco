# El Pollo Loco 🐔

> ⚠️ **Work in Progress** – This project is currently under development.

A jump-and-run game built with vanilla JavaScript and OOP.

## 🎮 How to play
- Arrow keys to move
- Space to jump

## 🛠️ Technologies
- Vanilla JavaScript
- HTML5 Canvas
- OOP (Classes, Inheritance)

## 📁 Project structure
- `js/` - Game logic
- `models/` - Game classes
- `img/` - Assets (property of Developer Akademie GmbH)
